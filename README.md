# sc-dictionary
[![License: CC BY 3.0](https://licensebuttons.net/l/by/3.0/80x15.png)](https://creativecommons.org/licenses/by/3.0/)

繁體+簡體中文詞庫字典檔

### 超齊百萬字典檔

* 檔案為 UTF-8 編碼
* 字典檔已經包含繁體中文與簡體中文字碼
* 字典檔為詞庫，目前有 1,109,412 個詞，不包含單中文字
* 格式：每一行一個詞，使用 \n (0x0A) 斷行
